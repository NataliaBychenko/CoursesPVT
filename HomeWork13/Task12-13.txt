Общее задание:
Создать следующие классы:
Элементы страницы:
1. Button
поля:
- name
- action
2. Field
поля:
- id
- value
3. Label
поля:
- name
4. Dropdown
поля:
- List&lt;String&gt; options
- boolean isSelected
- String name
5. Page
поля:
- ArrayList&lt;Button&gt;
- LinkedList&lt;Field&gt;
- HashSet&lt;Label&gt;
- HashMap&lt;Dropdown&gt;
- float id
- String title
Методы в классе Page:
Для каждой из коллекций в классе (4 шт) реализовтаь методы-обертки,
который будет принимать объект нужно типа и:
1. Добавлять в коллекцию
2. Удалять из коллекции
3. Заменять элемент в коллекции
4. Выводить все элементы коллекции

Создание всех объектов, их инициализация и работа методов демонстрируется
в метода main

2. Реализовать TreeSet в любыми сложными объектами (не стандартными)
